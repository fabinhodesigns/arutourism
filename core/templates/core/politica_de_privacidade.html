Política de Privacidade

Esta Política de Privacidade descreve como coletamos, usamos, e protegemos suas informações quando você utiliza o nosso site.

1. Coleta de Dados

Coletamos dados pessoais que você nos fornece diretamente, como nome, e-mail, e dados de navegação. Essas informações são usadas para proporcionar uma melhor experiência de navegação.

2. Uso dos Dados

Os dados são utilizados para personalizar a experiência no site, enviar notificações sobre atualizações e melhorar nossos serviços. Não compartilhamos suas informações com terceiros, a menos que exigido por lei.

3. Armazenamento de Dados

Os dados pessoais coletados são armazenados de forma segura, e só ficam disponíveis para os administradores do sistema para fins de suporte e melhorias do serviço.

4. Seus Direitos

Você tem o direito de acessar, corrigir, ou excluir suas informações pessoais. Para exercer esses direitos, entre em contato conosco através do nosso formulário de contato.

5. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão informadas diretamente em nosso site.

Se tiver dúvidas sobre esta Política, entre em contato conosco.
